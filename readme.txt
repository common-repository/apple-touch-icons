Sets the icons Apple Touch Icons for iPhone, iPhone Retina, iPad and iPad Retina for your web.

== Description ==

Easy form that allows to set the Apple Touch Icons for your web. It supports iPhone, iPhone Retina, iPad and iPad Retina icons.

It allows .mo and .po translation files. The plugin supports:

* English, Spanish.

== Installation ==

1. Unzip apple-touch-icons.zip.
2. Upload apple-touch-icons to your wp-content/plugins folder.
